📷 QR Code Trail

Scan a QR code at each spot to collect your stamp
How it works: Print these QR codes and place one physically at each attraction. When a visitor scans a code, the site checks their phone's GPS and only awards the stamp if they're actually standing at that spot — so scanning from home won't work. Points, stamps and badges unlock only on-site.

⚠️ To make scanning work on phones, host this file online (e.g. free GitHub Pages or Netlify) and set your address in SITE_URL near the top of the code. The QR codes below encode that address.
🧪 Testing: turn on test mode to try the reward flow from anywhere — check-ins will skip the GPS location check and pretend you're on-site. Remember to switch it off before going live.
